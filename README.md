# Understanding Random Forests: How the Number of Trees Affects Classification Performance

## Overview

This project demonstrates how Random Forest classifiers work and investigates how changing the number of trees affects classification performance.

The tutorial uses the Breast Cancer Wisconsin dataset from Scikit-learn and evaluates Random Forest models using different ensemble sizes.

This project was created as part of a Machine Learning coursework assignment.

---

## Objectives

The main objectives of this tutorial are:

- Understand how Random Forest works
- Compare model performance using different numbers of trees
- Visualise classification performance
- Analyse feature importance
- Discuss ethical considerations in machine learning

---

## Dataset

The project uses the Breast Cancer Wisconsin dataset available from Scikit-learn.

Dataset characteristics:

- Binary classification problem
- 569 samples
- 30 numerical features
- Medical diagnostic data

Dataset documentation:
https://scikit-learn.org/stable/modules/generated/sklearn.datasets.load_breast_cancer.html
